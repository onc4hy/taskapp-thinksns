	
#
# 安装说明
#

1. 安全起见, 请备份站点的所有文件和数据库
2. 解压文件并上传至服务器文件的apps目录中
3. 进入"管理后台 - 应用 - 应用管理 - 应用安装", 点击"待安装应用"中的"应用名称"为"task"的"安装"按钮以完成安装
4. 清理缓存
5. Enjoy
 
#
# 升级说明
#

1. 安全起见, 请备份task应用的所有文件和数据库
2. 上传最新版的应用并上传至服务器文件的apps目录中, 并覆盖原有文件
3. 进入"管理后台 - 首页 - 系统升级", 点击"应用名"为"task"的"升级"按钮以完成升级
4. 清理缓存
5. Enjoy
